# tmdb-tv-and-show-widget
A simple widget for displaying your TMDB watchlist dynamically
This, sadly, is not a copy-and-paste solution, but here are all the steps you need to take to get it working!
---
# Get Api Info
This whole project is basically built  on TMDB's Api. So you must make an account! (I also suggest putting the logo in your footer)
https://www.themoviedb.org/settings/api

### To get the ` Authorization: 'Bearer (get-token-from tmdb api)`
Go to your Api Settings Page https://www.themoviedb.org/settings/api
[![Screenshot-2026-02-23-at-11-30-41-My-API-Settings-The-Movie-Database-(TMDB)1.png](https://i.postimg.cc/W3jJCyvb/Screenshot-2026-02-23-at-11-30-41-My-API-Settings-The-Movie-Database-(TMDB)1.png)](https://postimg.cc/Z97RyVn1)
and copy your API Read Access Token

### To get the `const api_key = "get-from-tmdb-api)`
Go to your Api Settings Page https://www.themoviedb.org/settings/api
[![Screenshot-2026-02-23-at-11-30-41-My-API-Settings-The-Movie-Database-(TMDB)2.png](https://i.postimg.cc/MGw7JH0Z/Screenshot-2026-02-23-at-11-30-41-My-API-Settings-The-Movie-Database-(TMDB)2.png)](https://postimg.cc/fVHtxwg1)
and copy your API Key

### To get the `const account_id = "get-from-tmdb api)`
Go to Watchlist Movies https://developer.themoviedb.org/reference/account-watchlist-movies in the Api Reference 
[![Screenshot-2026-02-23-at-11-49-03-Watchlist-Movies.png](https://i.postimg.cc/9X1CGngg/Screenshot-2026-02-23-at-11-49-03-Watchlist-Movies.png)](https://postimg.cc/hhJknpBV)
And Copy Account Id!
